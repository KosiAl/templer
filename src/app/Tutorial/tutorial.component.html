Welcome to Templer.io

This website was made to help frontend developers faster develop boilerplate code. Currently it only supports creation of tables for Angular with angular material CSS library. But in the future it will support multiple frameworks and CSS librarys.

OK let's get started.

TEXTAREA EXCEL

In this textarea you can paste excel table. It also suports tables from other websites if there is proper formating on them.

1 / 7

CONVERT

Use this button to convert table from excel textarea to JSON structure.

TEXTAREA JSON

This textarea will produce JSON structure from your excel table. You can also paste JSON if you already have it and start generating templates. Currently, there is not support for nested JSON

3 / 7

EDIT JSON

Here you can edit your JSON structure. You can add and remove key-value pars. To newly created keys you can assign simple values like boolean, string or number. Data is generated automaticaly
